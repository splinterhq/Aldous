# Release Benchmarks

The individual `.json` files are exports from the visualizer after running each sample
without Kneedling or other filtering. These were used to help guide the initial values you'll
find in `tolerances.json` in the main test directory.

The `.html` file is the report from the `check_sensors.py` harness run over the samples
using the release model with no changes.

These are here for reference in case you ever need to know how far from the original your
installation might have become over time due to tuning.
