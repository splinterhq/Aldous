# Contributing 

Please see the CODE_OF_CONDUCT file first.

All ideas are welcome! Please open a Github issue as a discussion
for whatever you'd like to work on. We will take it from there!

## Things Needed Right Now

### More test samples we can distribute

See the README in `train/tests`. We need relatively well-known works 
(like political speeches, songs, poems and such) that are out of US 
copyright law that can be used as score targets for the adherence
and tolerance tests.

These samples have to be clear to distribute in the repo, and there
has to be at least searchable knowledge of how the text has impacted
others who have experienced it in the past. It's a common reference 
calibration point.

Obscure, out of copyright literary work isn't helpful for _these_
purposes, unfortunately.
